# Keyword Extraction using NLP

Implementasi **Keyword Extraction** pada dataset abstrak berbahasa Indonesia menggunakan beberapa metode **Natural Language Processing (NLP)** sebagai tugas mata kuliah **Natural Language Processing (NLP)**.

---

## Deskripsi

Repository ini berisi implementasi ekstraksi kata kunci (*Keyword Extraction*) dari dokumen teks berbahasa Indonesia. Proses yang dilakukan meliputi preprocessing teks, normalisasi, tokenisasi, stemming, hingga ekstraksi kata kunci menggunakan beberapa metode NLP.

Implementasi mengacu pada materi yang diberikan dosen serta artikel GeeksforGeeks mengenai Keyword Extraction dengan penyesuaian untuk bahasa Indonesia.
